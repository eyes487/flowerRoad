# 运行方法
1. 克隆代码：

```
git clone https://github.com/eyes487/huajiandao.git
```

2. 进入项目根目录并安装依赖：

```
cd huajiandao
npm install
```

```
数据库
在MySQL文件中运行 mye_q.sql 文件
```


3. 运行项目

```
npm start

在浏览器中输入http://localhost:8888/myE&Q.html

```
